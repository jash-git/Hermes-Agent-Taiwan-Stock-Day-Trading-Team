# SKILL: report-builder

## 使命 (Mission)
將 twse-free-daily 與 risk-filter 的結果，整理成清晰、專業且適合交易者快速閱讀的當沖報告。

## 職責範圍 (Responsibilities)
- 產出每日當沖建議報告
- 包含市場總覽、重點候選名單、個股簡析、風險提醒
- 支援多種格式輸出（Markdown、HTML、PDF）
- 記錄本次報告的資料日期與決策依據

## 可用工具 (Tools)
- `build_report.py`：報告生成器

## 輸入 (Input)
- 過濾後候選名單
- 原始市場資料摘要
- 當日市場總覽數據
- 可選：使用者偏好設定（重點類股、報告風格）

## 輸出 (Output)
- `Daily_Daytrade_Report_YYYYMMDD.md`（主要報告）
- `Daily_Daytrade_Report_YYYYMMDD.html`（美化版）
- 個股快照表格（含排名、關鍵指標、排除原因）

## 報告結構（建議）
1. **市場總覽**（大盤指數、成交量、漲跌家數、熱門族群）
2. **當沖機會評估**（今日適合當沖的程度）
3. **精選當沖候選名單**（Top 8~12 檔）
   - 股票代碼、名稱、排名分數
   - 前日收盤、漲幅、振幅、成交量、量比
   - 簡短理由
4. **風險提醒與注意事項**
5. **資料來源與更新時間**

## 關鍵原則
- 語言清晰簡潔，避免過度推測
- 強調「僅供參考，交易有風險」
- 版面乾淨，重點突出
- 保留完整審計軌跡（資料日期、使用的 skill 版本）

## 成功指標
- 報告可在 1 分鐘內生成
- 交易者能在 3 分鐘內抓到重點
- 格式固定，便於長期追蹤
